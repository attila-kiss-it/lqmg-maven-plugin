Usage

Dependency collection

No required dependency collection.

Plugin execution

The plugin requires some mandatory data that can be run. Futhermore must be set the plguin goals.

Minimal configuration

<plugin>
    <groupId>org.everit.db</groupId>
    <artifactId>lb2qd-maven-plugin</artifactId>
    <version>1.0.0-SNAPSHOT</version>
    <executions>
      <execution>
        <goals>
          <goal>generate</goal>
        </goals>
        <configuration>
          <sourceXML>${project.basedir}/src/main/resources/META-INF/changeLog/changeLog.xml</sourceXML>
          <packageName>com.example.lbqd</packageName>
        </configuration>
      </execution>
    </executions>
</plugin>

This case when builded the project generated JAVA sources to ${project.build.path}/target/generated-sources/lbqd/{packageName}. The sourceXML says where found the LiguiBase XML and packageName says to which package includes the classes.

Full configuration

<plugin>
    <groupId>org.everit.db</groupId>
    <artifactId>lb2qd-maven-plugin</artifactId>
    <version>1.0.0-SNAPSHOT</version>
    <executions>
      <execution>
	    <phase>generate-sources</phase>
        <goals>
          <goal>generate</goal>
        </goals>
        <configuration>
          <sourceXML>${project.basedir}/src/main/resources/META-INF/changeLog/changeLog.xml</sourceXML>
          <packageName>com.example.lbqd</packageName>
          <targetFolder>${project.basedir}/src/main/java</targetFolder>
        </configuration>
      </execution>
    </executions>
</plugin>

This case when builded the project generated JAVA sources to ${project.basedir}/src/main/java/{packageName}. The sourceXML says where found the LiguiBase XML and packageName says to which package includes the classes.

Configurations

A plugin configuration says to place of source XML, generated target folder, and the package name. Furthermore says plugin goals.

Name Description

phase

Says the build lifecycle phase. Default lifecycle phase: generated-sources.

goal

Says the execute goal. See more information Goals

configuration

Contains the plugin configurations (optional and required settings).

  • sourceXML
  • packageName
  • targetFolder
